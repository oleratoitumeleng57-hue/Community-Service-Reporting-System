# Community Service Reporting System

## Overview
This project is a Django-based web application designed to improve community service delivery by allowing residents to report and track issues such as water leaks, electricity faults, and potholes.

## Features
- Report service delivery issues
- Track issue status (Pending, In Progress, Resolved)
- User-friendly interface
- Simple navigation between pages

## Technologies Used
- Python (Django Framework)
- HTML/CSS

## System Structure
The system consists of:
- Report App: Handles issue submission
- Tracking App: Handles issue tracking

## How It Works
1. User submits an issue using the report page
2. The system generates a Report ID
3. User tracks the issue using the tracking page
4. Status updates are displayed to the user
